# Fydecor

FydeOS style window decoration plugin.

Forked from [decor](https://github.com/wayfireWM/wayfire/plugins/decor).

## Installation

```bash
git clone https://github.com/0xarch/wf-fydecor
cd wf-fydecor
meson build --prefix=/usr
ninja -C build
sudo ninja -C build install
```
